AWS Messaging application

The AWS Messaging application is an example application that uses Amazon SQS and other AWS services and the AWS SDK for Java version 2 API. Working with messages has never been easier. Just perform these steps:

  1. You can send a new message by choosing the Send Messages menu item. Select a user from the form, enter a message, and then choose Send.
  2. The AWS Messaging application stores the message in a FIFO queue. This queue ensures that the order of the messages is consistent.
  3. The AWS Messaging application polls the queue for all messages in the FIFO queue.
  4. The AWS Messaging application displays the message data in the view. The message body, user name, and an avatar are displayed.
  5. You can send and view multiple messages by using the AWS Messaging application.
  6. You can purge the queue.
